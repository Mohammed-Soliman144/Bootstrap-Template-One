# Bootstrap-Template-One

![Project ScreenShot](assets/img/link-preview-screenshot.png)

Interactive and Responsive Design with Image Slider and JS Functionality in each Page (Build Based on Bootstrap)

---

## Live Demo

**[View Live Demo Here](https://mohammed-soliman144.github.io/Bootstrap-Template-One/)**

---

## Features:

**1- Responsive Design:** Works seamlessly on different screen sizes such desktops, tablets and smart phones.

**2- Interactive UI:** Validation Form, Smooth Scrolling, Create Carousel mannually and other one by using Bootstrap.

**3- Animation UI:** In each Section has many animation work smoothly.

**4- JS Functionality:** Using Intersection Observer and Fetch to download any image from customize carousel when hosted in github server and Promise To switch betweeb iframe and fallback image automatically.

---

## Technologies Used:

**1- HTML5**
**2- Bootstrap Framework**
**3- CSS3**
**4- JavaScript (ES6+)**
**5- Font Awesome 7 Free Framework**
